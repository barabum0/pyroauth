# pyroauth
### A simple CLI that can create and authorize a session for pyrogram

## Installation
```shell
pipx install git+https://github.com/barabum0/pyroauth
pyroauth --help
```

## Usage
```shell
pyroauth \
    --id <API_ID> \
    --hash <API_HASH> \
    --output <ABSOLUTE_PATH_TO_YOUR_SESSION>
```
